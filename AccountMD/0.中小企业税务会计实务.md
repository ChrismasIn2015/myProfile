# 1.公司设立

```
✿：经济组织分类
	普通/有限/个人独资合伙企业
	无限/有限责任(常见)/股份有限/两合公司
✿：纳税人分类
	小规模/一般纳税人
********************************************
1.注册资本如何处理
	有限责任公司以认缴出资额为限对公司承担责任
	股份有限公司以认购的股份为限对公司承担责任
2.初始运营资金如何处理
	大额投资全部入账
		投资人借款/分红需要缴纳个人所得税(个人所得税)
	股债混合入账
    	无偿借款属于关联交易（调减纳税所得额）
3.技术投资会计处理：获得股权：股权转让属于增值税应税行为
	16/9/1后
	方法1：单位/个人直接入资：需开发票给受让对方
		技术入资需要有关部门认定-享受增值税免税优惠
		规定16/9/1后技术投资的收入可以选择递延至股权转让时纳税
		"借长期股权投资/贷无形资产/营业外收入(溢价部分)"
	方法2：单位/个人售卖入资：需开发票给受让对方
		签订技术转让合同取得资金，再进行投资
		单位/个人技术转让所得<500部分免征企业所得税/>500减半征收
4.技术受让方会计处理：股权转让时需开发票/评估报告
	允许无形资产摊销，并且进行税前扣除
5.各投资人股权如何分配：公司章程是规范股东权利义务的最高准则
	1.按出资比例确定分红比例(以章程为准)
	2.按出资比例确定表决权比例
	3.破产时，在支付所有费用后按出资比例分配剩余财产
6.异地设立公司经营享受税收优惠
	成本费用主要由原公司承担(如异地报销)
	收入主要由异地公司承接：签订转包合同(原价)
	这种操作等同于关联交易，被稽查需要求纳税调整
```

# 2.销售业务

```
销售业务
    1.产品(17%)+软件(3%)：分别核算以软件为主：高科技产品企业
    2.产品(17%)+服务(6%)：分别核算以服务为主：科技教育培训企业
    3.多服务模式：
    	融资租赁企业(租赁收入17%+服务收入6%)：高毛利，压低了租赁定价
    4.销售固定资产：不含不动产
    	进行优惠简易计税：按3%降低至2%纳税，开局普通发票，同时进行进项税转出
    	放弃优惠：正常按17%进行纳税
销售模式
	1.产品赠产品模式：视同销售，需缴纳增值税/企业所得税，计入销售费用
	2.有偿售后服务：视同销售，需缴纳增值税/企业所得税，计入销售费用
	3.服务赠产品模式：视同销售，需缴纳增值税/企业所得税，计入销售费用
	4.服务赠服务模式：视同销售，需缴纳增值税/企业所得税，计入销售费用
	5.商业折扣：需在同一发票下列支/按折扣后销售额纳税
	6.消费积分：需在同一发票下列支/按折扣后销售额纳税
	7.消费红包：需在同一发票下列支/按折扣后销售额纳税
关联交易，定价不合理
拓宽销售面，发展销售渠道：代理商/经销商
	自然人代理商：以发票报销支付佣金(违法)
	资质代理商：
		1.买断模式：按差价获利
		2.佣金模式：按销售量获利
		对于渠道商来说佣金超过5%部分不得扣除
		可以转换为购买服务模式全额扣除
```

# 3.采购业务

```
税负由买方全部承担：合法
	约定他人负担税负可以免除自己的纳税义务/若无需自行依法纳税
    本公司(买方)按合同约定向纳税义务人支付全部税费后，如果责任方未按时纳税，导致出现滞纳金、罚款，由责任方（纳税义务人）自行负担；无论何种原因导致纳税义务人少缴税款，事后被税务机关追缴，该追缴部分由责任方（纳税义务人）自行负担；无论何种原因导致责任方（纳税义务人）多缴税款，事后被税务机关退回，该退回 部分应在责任方（纳税义务人）收到款项之日起若干个工作日内返还本公司。
    
1.购买二手房屋时
	负担卖方税赋：签订税费转嫁合同
2.租赁个人房屋
	负担出租个人方税赋：签订含税价租赁合同
3.租赁员工车辆
	签订租赁协议
	不超过三万/月部分，个人不用缴纳增值税
	不超过四千部分，扣减八百(扣减20%)按20%缴纳个人所得税
4.支付餐饮服务费
	60%：对外招待费(不超过销售额5%)：为公务/业务活动需要合理开支的接待费用(交通/用餐/住宿)
	15%：对外业务宣传费：礼品
	对内差旅/会议（小额调整）：不超出规定日标准属于相关费/超出即为招待费
	14%：对内商业保险/福利费：允许税前扣除
		不包括工资奖金/教育经费/社保公积金/补充养老医疗保险
	2%：公会经费
	2.5%：教育/培训经费
	# 旅游费用算入职工福利费/专票不建议抵扣进项税
	
5.个人非生产经营报销计入红利所得计征个人所得税（公款报销）
```

# # 关于发票

```
无效发票
	1.假发票
	2.抬头是其他公司/个人不完整/无纳税人识别号-统一社会信用代码
	3.当期列支的成本/费用未取得发票
	4.跨期发票
不相关发票不得税前扣除
	1.真实发票业务为非生产经营相关
开具增值税发票
	1.小规模纳税人：税务机关代开/部分可自行开具增值税发票
		住宿业-鉴证咨询-建筑业
	2.不得开具增值税专用发票
		向个人进行销售
		销售固定资产/旧货/零售
		自然人应税行为
进项税抵扣
	1.专用发票/海关增值税缴款书/农产品发票
	2.路桥闸通行费
```
