## 民法制度

### 1.物权法

```
【物权法】
	1.物权变动
        不动产登记制度-物权法第9条
            不动产物权的设立变更转让消灭，经依法登记而发生效力;
            未经登记不发生效力;但法律另有规定的除外。
            依法属于国家所有的自然资源，所有权可以不登记。
        动产交付-所有权转让
	2.所有权
		所有权分类：国家/集体/私人所有权
		一/建筑物区分所有权(业主)
			专有部分专有权
			共有部分共有权/管理权-业主大会-委员会
		二/相邻关系
		三/共有权：按份共有/共同共有-婚姻
		四/用益物权
			土地承包经营权：
				林牧农渔：可转包互换转让等
			建设用地使用权（使用权期）：
				出让：招标/拍卖/协议
				划拨：按程序提出申请/无交付费用/限制用途
			宅基地使用权
				村民依法在集体所有土地上占有所有自有住宅/附属设施
			地役权：
				使用他人土地
			担保物权：
				债权人优先享有债务人的担保物清偿
				一/抵押权：债务人提供财产(不转移物权)担保债务履行
				二/质押权：债务人出质财产(转移物权)担保债务履行(公示主义)
				三/留置权：债权人留置合法占有的债务人动产
```

### 2.代理制度

```
【代理制度】
	概念：民事行为：委托/法定/指定代理
		指代理人在代理权限内，以代理人的名义与第三人实施法律行为
		由此产生的法律后果直接由被代理人承担
```

## 商事法律制度

```
# 企业 → 公司(大资金)
1.合伙企业：
	指两个以上民事主体共同出资经营共付盈亏
	一/普通合伙企业 
		合伙人对企业债务承担无限连带责任
		特殊的普通合伙企业：事务所-非企业专业服务机构
	二/有限合伙企业：2-50人
		由有限合伙人和普通合伙人共同组成
		普通合伙人承担无限连带责任/有限合伙人以认缴出资为限承担责任
2.个人独资企业：
	投资人以个人财产对企业债务承担无限责任
3.公司法：
	具有独立的法人地位：无限/有限责任/股份有限/两合公司
	指股东依照公司法规定,股东以其认缴的出资额
	或认购的股份为限对公司承担责任
	一/有限责任公司：50以下股东：股东会 →董事会 →监事会
	二/股份有限公司：2-200人：招股认购
	# 公司会计制度
		财务会计报告
		公积金制度：用于弥补亏损
			1.利润-盈余公积金
			2.股票-资本收益-资本公积金
	# 公司合并/分立/变更/解散/清算
		合并：吸收合并(ABX)/新设合并(AXBXC)/收购&兼并(AB)
		清算：普通-股东/特别-法院/破产-法院&债权人(强制)
4.外商投资企业法
	中外合资/中外合作经营/外资
5.破产法：全部财产进行接管清算
6.合同法
	买卖/供用资源/赠与/借款/租赁/融资租赁/承揽/建设工程
	运输/技术/保管/仓储/委托/行纪(信托)/居间
	一/订立方式：
		邀约：希望和他人订立合同
		承诺：接受邀约,承诺生效时合同成立
	二/债权人要求担保
		保证：第三人为债务人债务履行担保
		抵押/质押/留置：财产担保
		定金：货币担保
```



## 其他法律制度

```
1.反不正当竞争法
	欺骗性标志/商业贿赂/虚假宣传/侵犯商业秘密
	不当有奖销售/诋毁商誉
2.反垄断法
	禁止垄断协议
	禁止滥用市场支配地位
	控制经营者集中：企业合并需要申报审查
3.消费者权益保护法
4.产品质量法
5.税法
	企业所得税
	个人所得税
		一/工资薪金
		二/个体工商户
		三/对企业事业单位承包经营.承租经营所得
		四/劳务、稿酬、特许权使用费、利息股息、
			财产租赁、财产转让、偶然、其他所得
	增值税
		一/一般纳税人：
			应税销售/劳务占比>50%，销售额超过50万
			批发零售经营销售额>80万
		二/小规模纳税人：采用简易办法征收(3%税率)不允许抵扣进项税
			应税销售/劳务占比>50%，销售额不超过50万
			个人/不经常发生增值税行为的企业
		三/税务管理
			向工商行政管理机关登记本单位注册信息
			自领取营业执照之日起30日内，向税务机关申报办理税务登记
6.证券法
	证券交易所：股票/基金/债券
	证券经营机构
		接受委托发行证券
		代理证券买卖业务
		代理证券股息股利支付
7.票据法
	委付汇票/自付本票/委付支票
8.会计/审计法
9.保险法
10.社会保障法
	社会保险：基本养老/疾病/工伤/失业/生育
	住房福利：住房公积金/廉租房/经济适用房/住房金融
	劳动法/劳动合同
```
