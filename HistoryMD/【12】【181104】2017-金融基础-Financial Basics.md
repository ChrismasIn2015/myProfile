# 金融基础

## 1.货币

```
起源：提升交换效率
本质：能与任何商品交换的特殊商品
历史：
	1.实物
	2.金属
	3.代用货币（本票等）：有十足兑换信心
	4.信用货币：不兑现制-黄金
		- 以银行/国家信用作为担保
		- 仅持有一部分现金，其余为记账符号
	5.电子货币：未来
制度：
	- 银本位：最早
	- 金银复本位：不稳定：劣币驱逐良币(偏好保存良币)
	- 金本位
		= 第一次世界大战冲击了金本位制度
		= 1933世界经济危机：黄金紧缺，金本位奔溃
		= 布雷顿森林体系：美元/会员国=黄金
	- 不兑现信用货币制度 
```

## 2.信用 & 利率

```
信用概述
	交易主体/目标客体/约定工具/约定时间/约定利率
关于信用工具
	- 承兑汇票/本票/支票
	- 信用卡
	- 信用证（国际贸易用）
	- 股票：获取股利
	- 债券
	- 国库券
	- 大额可转让定期存单
金融衍生工具
	- 远期合同：合同双方约定未来价值
	- 期货合同：由交易所，交割未来价值
	- 期权合同/互换合同
1.利息/利率的成因
	# 影响深远，意义重大；负利率时代有什么影响？
		1.家庭财富多元化
		2.政府债务积极化解
		3.影响资产市场：房地产等
		4.影响企业投融资成本
	1.社会平均利率
	2.借贷供求
	3.货币政策
		- 扩张性货币政策：基准利率下降
		- 紧缩性货币政策：基准利率上升
	4.经济周期
		- 经济繁荣：基准利率上升，抑制投资
	5.通货膨胀
		- 货币贬值，实际利率下降
	6.国际利率
		- 资本内/外流
2.利息/利率的影响（利率上升）
	1.宏观经济：提升储蓄，抑制消费
	2.资金供求：供小于求
	3.投资：储蓄大于投资
	4.货币供应：信贷成本提升，信贷规模降低，货币供应量下降，物价下降
	5.国际收支：吸引外资抵消逆差
# 降息对房地产的影响：1.货币价格高/资金压力大，促进购房潮
```

## 3.金融机构

```
# 金融机构是经济体系中最复杂敏感的部分，是经济系统中的关键部门
1.监管机构
	国家性权力机构
2.经营机构
	1.存款性质机构：商业银行等
	2.非存款性机构：不得吸收公众储蓄存款，接受契约型资金
3.功能
	1.信用中介，促进资金融通
	2.支付中介
	3.降低交易成本
	4.处理信息不对称
	5.转移和管理金融风险
4.中国的金融机构体系
	# 高度集中的金融机构体系
	1.中国人民银行/银监会、证监会、保监会
	2.商业银行（国股城商农商信用社）/政策性银行/非银行金融机构
	3.非银行金融机构
		1.金融资产管理公司：华融/长城/东方/信达：专门处置国有独资不良资产
		2.证券机构：
			证券公司（券商）：承销债券/代理买卖/参与企业兼并收购（目前百家）
			证券交易所：上海/深圳
			证券登记结算公司
		3.保险公司
		4.信托公司：按照委托人指定的目的和要求运用资金财产（68家）
		5.基金公司：聚集中小投资人资金（107）
		6.财务公司：企业内部集资组建，服务自身企业
		7.金融租赁：38家
		8.贷款公司：服务农村
5.国际金融机构
	1.国际货币基金组织-华盛顿
	2.世界银行集团
		- 世界银行
		- 国际开发协会
		- 国际金融公司
		- 多变投资担保机构
	3.区域国际金融机构
		- 国际清算银行：国际清算组织
		- 亚洲开发银行：东京
		- 亚洲基础设施投资银行
```

## 8.通货膨胀/紧缩

```
1.度量通货膨胀
	CPI-消费者物价指数
	WPI-批发物价指数
	GNP-国民生产总值平减指数
	1.产出效应：实际经济增长率低于潜在经济增长率，于是适度扩大货币发行，增加投资性支出，最终增加了劳动供给
	2.收入分配效应：货币贬值下的价格变动
2.通货膨胀成因
	1.需求拉升型
	2.成本推动型
3.通货膨胀治理
	1.紧缩财政政策
		税收/削减财政支出/发行公债
	2.紧缩货币政策
		减少货币供应/提高利率
		刺激生产力：降税/削减开支/限制货币发行
```

```
# 全球大萧条与通货紧缩：较少
1.价格全面下跌
	供给过剩
	需求过低
2.财富缩水
	大失业：工资下跌
	大萧条：利润降低
3.形成原因：极为复杂
	1.债务通货紧缩论：过度负债风险
	2.信贷过度扩张论：生产过剩
	3.资产缩水论：资产泡沫破裂，社会投资降低
4.治理对策
	1.货币政策：刺激投资消费
	2.财政政策
		加大政府支出/减税/增加公债
	3.总论
		增加有效需求：投资消费出口
		调整产业结构：增加有效供给
		深化改革：适应环境
```
