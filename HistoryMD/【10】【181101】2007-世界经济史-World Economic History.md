### 6.商业革命→工业革命(18世纪后期)

```
#商业化：经济各个部门参与市场买卖进行互动，以市场需求为中心，以获利为目的，直接动力来源于日益扩大的市场需求，而市场需求来源于人口增长的压力
● 欧洲-不适宜的农业条件 & 商业阶层兴起
● 葡萄牙-西班牙：非工业消耗过多，工业未兴起
● 英法荷德：渐渐具有工业革命趋势
	荷兰阿姆斯特丹交易所
	英国圈地运动（高货币价格使政府支持占有已(未)租赁土地，扩大贸易，为资本主义提供自由劳动力）
● 货币增多及支付清算需求 - 金融业兴起（银行汇票//信贷/证券）
● 重商主义兴起，高流动性的中产阶级出现，重视科学大于重视宗教（禁欲禁利）
● 资本不断积累，以利润的形式流入欧洲，对机器的需求引发工业革命
```

### 14.战争 - 残酷的平衡

```
# 社会主义是一种工业化模式 - 战争是时代的选择

1.二十世纪初第一次世界大战1914-1918
	1.1 【主要原因】是资本主义大国殖民地争夺 
		- 经济空间/资源竞争/有利战略点
	1.2 美国贸易顺差激增-黄金储备世界30%
# 金本位制度：以货币中的黄金含量为汇兑依据进行贸易
# 货币面值也许不一样，但同价值商品黄金总量是一样的
	1.3 一国国际收:出口货物换回黄金，支:支付黄金进口货物 = 平衡
	1.4 一国贸易逆差：进口>出口，黄金储备降低，逐渐稀有，货币流通性降低，外汇汇率(外/内)面值上涨（N++：1）进口货物的逐渐增多，物价逐渐降低，国际市场竞争力提高，贸易收支恢复平衡。
	1.5 一国贸易顺差：进口<出口，黄金流入，货币流通增多，物价上涨，外汇降低（N--:1）,国际市场竞争力降低，出口减少，恢复平衡。
	1.6 但是现实中贸易逆差往往通过国家调高资本利率（银行利率）/降低贴现利率，引起套利资本入场来保持黄金-货币量/而处于贸易顺差的工业国家在发展中国家大力投资，通过输出货币保持物价。建立世界货币信心的同时，超发货币的弊端往往转嫁到其他国家。
############################################################
2.1825十九世纪初英国爆发第一次经济危机，后资本主义国家每隔10年爆发一次危机，尤其以1857，1929-33大危机。
	2.1 美国获利，成为世界大债权国/金融中心 
		- 经济（空间）扩张使投资者信心满满，股市持续上升
	2.2 
		德国遭受的疯狂报复掠夺
		-《凡尔赛条约》高额赔款/货币贬值
		- 1933年希特勒上台进行凯恩斯经济振兴
		美国罗斯福新政：国家干预
		- 1929银行/证券大破产，工人大失业
		- 1936凯恩斯《就业、利息和货币》
		日本-错误经济政策
		- 1937侵华战争
		- 1941太平洋战争爆发
	2.3 二战：资本主义国家之间的市场/商品销售争夺
	2.4 丘吉尔：铁幕演说-冷战开始
```
