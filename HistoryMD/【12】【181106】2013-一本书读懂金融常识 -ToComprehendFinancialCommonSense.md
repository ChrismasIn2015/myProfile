# 金融常识

## 常识

### 1.货币

```
1.货币概念
2.货币制度
3.供给表示：M0M1M2
	通过三大指标，来判断流通货币量是否充足
	M0：居民消费
	M1：企业居民资金松紧变化
	M2：社会总需求/未来通货膨胀：货币投放状况
	- 通过M1/M2增长率来反映经济扩张
4.货币流通速度对经济的影响
	1.周转速度：流通次数X商品总价格：通货膨胀
5.虚拟货币：商品
6.货币需影响因素：货币是最灵活的资产
	- 收入水平
	- 利率
	- 商品价格
	- 货币流通速度：一定时期内货币转手次数
```

### 4.汇率

```
# 汇率是国际经济交往至关重要指标
1.影响汇率的因素
	- 进出口差额：出口＞进口：贸易顺差，资本流入，汇率下跌（N--：1），货币升值
	- 资本流动/利率差异/通货膨胀
2.汇率研究的基本问题
	- 汇率根据什么来决定
	- 汇率制度确定
	- 汇率政策的分析
3.汇率变动对经济的影响
	- 问题1：通货膨胀率高于美国/人民币却升值
	- 问题2：人民币汇率上升/中国却存在贸易顺差"资本流入"
	# 外汇储备作何用途
	# 物价变动/进出口贸易/资本流动
4.汇率政策工具
	- 维持对外价值稳定
	- 维持对内价值稳定：使国内购买力不受外来扰动（资本）影响
```

```
# 聚集闲置资金
1.股票
	宏观
		- 导向资金流动
		- 经济信息传递
		- 宏观调控重要手段：法定存款准备进率/再贴现率/公开市场业务

		- 融资

	类型：无形/有形：证券交易所
	功能
		- 货币买卖：企业个人团体/外汇清算：银行/外汇转换/外汇信贷
	作用
		- 购买力转移/借贷融通/为金融活动提供便利

	由来：现货交易 与 远期交易日期的矛盾

6.债券市场
	一级市场/发行市场

		银行间/商业银行柜台/证券交易所
```

### 7.金融危机

```
1.金融风险
	信用/市场/操作风险/信誉
2.影响因素
	经济周期/成本转嫁到了金融机构/信息不对称/历史信息规律/泡沫
# 次贷危机
	即次等级抵押贷款机构破产：美国08年房地产
3.泡沫经济
	# 1980+日本股票/房地产
	金融投机过度
	原因：流动资金充裕/约束机制缺乏
4.经济下行周期
	短周期：3-4年
	朱格拉中周期：9-10年：国民相关
	康德拉季耶夫长周期：50-60年
	库兹涅茨周期：20年：建筑业
	熊皮特：创新周期
```
